# Disk Packing Visualizer

A browser-based research tool for experimenting with equal-radii hard disk packings. This application allows users to load JSON files describing contact classes, browse them visually, and interactively manipulate the disk configuration while enforcing hard-disk constraints.

## Setup Instructions

This project is built using Next.js and React.

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Run the development server:**
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
