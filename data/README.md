# Data Files

This directory contains JSON files with contact class definitions for disk packings. 

*   **`3disks.json`**: Configurations for 3-disk packings.
*   **`4disks.json`**: Configurations for 4-disk packings.
*   **`5disks.json`**: Configurations for 5-disk packings.
*   **`6disks.json`**: Configurations for 6-disk packings.

**Note:** Files in this directory are read-only inputs. The application must never write to or overwrite them.
